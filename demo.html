This is demo page with javascript for adding comments in the static page

In a web page with static HTML content inside, user should be able to click anywhere on the page and add a comment. When page is refreshed, the annotations should be preserved. User should be able to edit and delete the comments and annotations.

Single click functionalities

  • Click anywhere - open form for add comment
  • Click and hold the mouse on green point - you can drag point

Double click functionalities

  • Double click point -it will open existed comment and you can edit/delete the comment
  • Double click form area or textarea - the form will close
